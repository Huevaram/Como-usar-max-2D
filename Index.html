Que es max 2D

Todo lo que necesitas para llevar tu idea desde el concepto hasta el juego terminado. ¡Max2D Game Engine es el entorno de desarrollo de juegos 2D definitivo para dispositivos móviles!. Con esta aplicación vas a poder crear videojuegos sin la necesidad de saber programar es rapido y facil de entender aparte de que ya viene incluido con algunos controles para hacer la programación de un videojuego muy facil.



Como poner una plataforma

Primero debes elegir la plataforma que desean, luego con el cubo que esta arriba.

Luego le damos a sprite y seleccionamos la carpeta para poder elegir nuestra plataforma.
Cuando hallamos seleccionado nuestra plataforma, le damos click a los botones del lado derecho iniciando por el de abajo, una ves que hemos hecho esto nos aparecera un menu y vamos a poner en transform para ajustar el tamaño y despues vamos a seleccionar +add component y seleccionamos rigid body, le damos a body type y le ponemos static, seleccionamos ahora box colider y lo ajustamos hasta que este de acuerdo a la plataforma y listo.

Como poner un personaje

Para poner un personaje es lo mismo que la plataforma le damos a sprite y seleccionamos la carpeta para poder elegir nuestro personaje.
Le damos click a los botones del lado derecho iniciando por el de abajo, una ves que hemos hecho esto nos aparecera un menu y vamos a poner en transform para ajustar el tamaño y despues vamos a seleccionar +add component y seleccionamos rigid body, le damos a body type y le ponemos static, seleccionamos ahora box colider y lo ajustamos hasta que este de acuerdo al personaje y listo.

Como hacer que se mueva el perdonaje.

Ahora vamos a ver como hacer que se mueva el personaje.
Primero seleccionamos a nuestro personaje, Ahora le vamos a dar al boton de arriba para seleccionar Ui Components y le damos a joystick despues nos vamos a global variable y seleccionamos number, le cambiamos el nombre y en donde dice show debug le ponemos true, luego le damos al script del personaje y seleccionamos On-joystick y set-velocity y lo unimos y ponemos en Event:direction change y en variable: joystick, en donde dice linear-X pon Expression_solver y escribes lo siguiente (joystick_value_x * joystick_distance * deltatime * 30000)
